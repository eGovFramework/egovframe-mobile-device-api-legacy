# egovframe-mobile-device-api-legacy

**이 eGovFramework 4.3.x 이하 버전의 Legacy 저장소입니다.**

eGovFramework **5.0.0 이상**을 사용하신다면 아래 저장소를 참고해 주세요.

[egovframe-mobile-device-api](https://github.com/eGovFramework/egovframe-mobile-device-api)

 ---

## 개요

Android 및 iOS 모바일 애플리케이션 예제를 제공하는 저장소입니다.

## 지원 버전

| eGovFramework 버전 | 저장소 |
|---|---|
| 4.3.x 이하 | 이 저장소 (Legacy) |
| 5.0.0 이상 | [egovframe-mobile-device-api](https://github.com/eGovFramework/egovframe-mobile-device-api) |
